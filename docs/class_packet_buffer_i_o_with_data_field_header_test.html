Define a packet of verification service acceptance success (subtype 1) that is as follows:

(service type 13 is not verification service, but keep it for now) version type dfh appid sequence_control length ccsds pus_version ack type subtype error 0 0 1 1, 2 (0b11, 2) 9 0 1 0 13 1 794

length = 9 - 1 = (ccsds, pus, ack, type, subtype)(3 octet) + (app. id)(2 octets according to ECSS-70-E-41A) + (sequence control)(2 octets) + (error control)(2 octets) + (-1)

Sequence control contains flags and count. flags = 0b11 means stand_alone. Then count = 1.

error control = 0 because I won't use a CRC8 algorithm during unit testing.
